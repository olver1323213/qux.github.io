Help

  • !add_our_bot - Add our Discord bot to your server.
  • !addcolor <color> - Add a color role to yourself.
  • !addrole <role> - Add a role to a user.
  • !announce <message> - Announce a message in the server.
  • !ban <user> - Ban a user from the server.
  • !change_channel_name <new_name> - Change the name of a channel.
  • !change_topic <new_topic> - Change the topic of the current channel.
  • !clearchat - Clear Chat
  • !commands - Display available commands.
  • !createrole <role_name> - Create a new role.
  • !creator - Display information about the server creator.
  • !deleterole <role> - Delete a role from the server.
  • !help - Display help information.
  • !hi - Greet the bot.
  • !kick <user> - Kick a user from the server.
  • !make_channel <channel_name> - Create a new text channel.
  • !move_to_vc <user> - Move a user to a voice channel.
  • !ourinvite - Get the invite link for our server.
  • !punishments - Display server punishments.
  • !removerole <role> - Remove a role from a user.
  • !rules - View server rules.
  • !slowmode <time> - Set slow mode for the channel.
  • !timeout <user> - Timeout a user from the server.
  • !unban <user> - Unban a user from the server.
  • !website - Get the link to our website.
  • !ytflaek - Get Flaek YT link
